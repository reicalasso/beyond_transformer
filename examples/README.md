# PULSE + MaaS Examples

## Quick Start

Run any example:
```bash
python examples/maas_basic_usage.py
python examples/maas_advanced_usage.py
python examples/maas_integration_demo.py
```

## Examples

### 1. Basic Usage (`maas_basic_usage.py`)
Demonstrates fundamental operations:
- Write memory
- Read memory
- Update memory
- Delete memory
- Get statistics
- Consolidation

### 2. Advanced Usage (`maas_advanced_usage.py`)
Shows advanced features:
- Dynamic routing
- Memory consolidation strategies
- Query expansion
- Multi-layer querying
- Memory decay simulation
- Batch processing

### 3. Integration Demo (`maas_integration_demo.py`)
Complete chatbot simulation:
- User onboarding
- Active conversations
- Memory consolidation over time
- Natural forgetting
- Real-world use case

### 4. Server Example (`maas_server_example.py`)
HTTP API usage:
- Start server instructions
- Test all endpoints
- Example requests

## Requirements

```bash
pip install torch
```

For server examples:
```bash
pip install flask flask-cors
```
